# FPS TEMPLATE

A simple FPS template created in Unreal Engine 5 using C++ and Blueprints.

The project includes an aim training ground with a player character, a weapon, and two interactive buttons:
Start button - begins spawning targets on the wall.
Stop button - stops target spawning.

This project is designed as a starting point for larger FPS projects.
